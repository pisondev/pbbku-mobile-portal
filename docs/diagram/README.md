# Diagram Laporan PBB-Ku

Folder ini berisi aset diagram untuk laporan akhir.

## File

- `2-1_arsitektur-sistem-pbbku.png`: diagram arsitektur existing.
- `sequence-login-otp.mmd`: sequence login NIK dan OTP simulatif.
- `sequence-search-detail-sppt.mmd`: sequence pencarian objek pajak sampai detail SPPT/tunggakan.
- `sequence-report-draft.mmd`: sequence laporan perubahan bangunan sebagai draft lokal.
- `sequence-reminder-payment.mmd`: sequence pengingat jatuh tempo pembayaran.
- `api-request-flow.mmd`: flow endpoint API internal kompatibel SIMPBB.

File `.mmd` memakai Mermaid dan bisa dirender ke PNG/SVG untuk dimasukkan ke laporan atau presentasi.
